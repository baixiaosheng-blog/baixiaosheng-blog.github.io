    Android studio Glide v4框架导入

    目前Glide最新版本为 4.11.0

    Glide中文文档网址：https://muyangmin.github.io/glide-docs-cn/doc/download-setup.html

    中文文档有详细的官方介绍

    以下简述导入方法

    打开Grade文件，添加依赖代码：

    implementation ‘com.github.bumptech.glide:glide:4.11.0’

    然后，点击右上角“Sync Now”
